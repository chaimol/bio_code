###### 用法：
- farename 用于重命名fasta文件的头部，支持按指定新文件的命名或指定前缀+数字命名
- commerge 是练习shell编程
- chr 主要是增加或删除bam文件的chr
- getgene 用于从gtf/vcf文件提取某条染色体或某个区域中所有的基因的gtf/vcf
- pathadd 用于把指定路径添加到环境变量。
