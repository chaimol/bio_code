# Features

for exchange fasta file head. can be use a new head fasta file or a prefix head.

# Install

`chmod 757 farename`

`./farename -h`



# Usage example

`./farename -f demo.fa demo_new_head.fa output.fa`

`./farename -p demo.fa GRCh19 output.fa`
